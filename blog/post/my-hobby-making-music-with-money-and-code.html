My Hobby: Making Music with Money and Code

Posted on March 9, 2024

Guess what? I've been mixing up my love for cool beats with something a bit different – trading digital coins, also known as cryptocurrencies, and all with the help of a little coding. I want to share with you how I do it, so if you’re curious, here’s a super simple guide.

Starting Out: Coding to Trade Digital Money

So, there's this thing called Gemini – it's like a digital wallet for your online coins (like Bitcoin). And the cool part? I use Python (that's a super simple coding language) to tell Gemini when to buy and sell coins for me. Think of Python as my personal DJ that knows exactly when to spin the tracks.

Setting Up Gemini and AWS Lambda

The Magic Box of Gemini Python Library

The gemini-python library is a toolbox that lets your Python code talk to Gemini and make those buy and sell orders. It’s as if you're using a fancy DJ mixer to control your music, but for trading!

Before you start trading for real, you can practice with fake money on Gemini's Sandbox. It’s like doing a rehearsal before your big music show.

Here’s the best part: you can set up your Python code to run on a schedule, automatically, just like a playlist. If you want to buy Bitcoin every week, you can tell Python to do it for you. It's like having your own robot that buys coins for you while you chill.

Let’s Wrap It Up

Combining music, coding, and trading is like creating a unique song. If you're just starting, don't worry if it feels like learning a new instrument – play around, have fun, and before you know it, you'll be making your own kind of music with your trades. Catch you later, and happy trading!
